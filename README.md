# HTML Authentication POC

This project contains simple HTML authentication pages created using only HTML.

No CSS or JavaScript is used. Only HTML structure and anchor tag redirections are implemented.

## Pages Included

1. login.html
   - Login page with links to Register, Forgot Password and Dashboard.

2. register.html
   - Registration page with link to Login page.

3. forgot-password.html
   - Forgot password page with link to Reset Password.

4. reset-password.html
   - Reset password page with link to Login page.

5. dashboard.html
   - Dashboard page with Logout link to Login page.

## Features

- Pure HTML pages
- Anchor tag navigation
- Proper redirections between pages
